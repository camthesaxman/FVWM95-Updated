OnyxFileDialog

OnyxFileDialog objects are used to invoke the standard File Opend/Save dialog box.

Methods:

  • virtual int Create()
    Creates the dialog object. Sets the file type description to "All Files" and the pattern to "*".
  • virtual void Show()
    Shows the dialog box. The function does not return until the user closes the dialog.
  • virtual int AddFileType(char *Description, char *Pattern)
    Adds the specified description and pattern strings to the existing file type list.
  • virtual void ClearFileTypes()
    Clears the file type description and pattern array.
Properties:

  • const OXWindow *DisplayWindow
  • int DialogType
    Either OnyxOpenFileDialog or OnyxSaveFileDialog.
  • char DirectorySelected[255]
  • char FileSelected[255]
  • int Cancelled Upon return from Show(), this variable is set to True if the user clicked on the Cancel button to close the dialog box. Otherwise it is set to False.
Events:

None.
