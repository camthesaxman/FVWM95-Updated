OnyxFontDialog

OnyxFontDialog objects are used to invoke the standard Font selection dialog box.

Methods:

  • virtual int Create()
    Creates the dialog object.
  • virtual void Show()
    Shows the dialog box, the initial font selection is set to the Font property. The function does not return until the user closes the dialog.
Properties:

  • const OXWindow *DisplayWindow
  • int Cancelled
    Upon return from Show(), this variable is set to True if the user clicked on the Cancel button to close the dialog box. Otherwise it is set to False. Note: Currently not supported and always set to False.
Events:

None.
