OnyxBase

Not really an object, but a collection of global functions and variables that are used by OnyX applications.

Functions:

  • void Initialize(char *ApplicationName, int argc, char *argv[])
    This method performs basic application initializations. Among other things, it opens the connection to the X display and creates the CentralMessageCenter object. Command line arguments are passed through argc and argv. Currently only the -display display_name option is recognized.
  • const OXWindow *ApplicationRoot()
    Returns the application's root window.
  • const OXWindow *Application()
    Same as ApplicationRoot().
  • OXClient *ApplicationClient()
    Returns the application's OXClient. OXClient is an xclass object that establishes the connection to the X display, manages resources and dispatches events.
  • OPicturePool *ApplicationPicturePool()
    Returns the application's "Picture Pool". Xclass has a sharing mechanism that allows efficient usage of X resources. ...
  • void ApplicationRun()
    This function calls the main message loop.
  • void ApplicationClose()
    Causes the application to close all open windows and exit.
Variables:

  • MessageCenter CentralMessageCenter
    The CentralMessageCenter object receives messages from all OnyX objects and dispatches them to the appropiate handling functions. Application programs usually do not have to deal with this object.
